# Project Title

Password Generator

## Description

Password generator application.

## Getting Started

### Dependencies

* Golang

### How does it work?

* The project can be run via terminal/cmd.
```
go build pass_gen.go
main
```
or
```
go run pass_gen.go
```

</br>

## Project Features

* It can create passwords containing alphabets and special characters.
